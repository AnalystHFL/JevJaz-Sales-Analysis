# Sales Analysis of JevJaz Kids Store (Explanatory Dashboard creation using MS Excel)

## Project Objective
The JevJaz Kids Store wants to create a clear and simple monthly sales report for October. This will help the store owner easily understand how customers are shopping and which products are most popular. With this information, they can make smarter decisions to grow sales and improve customer satisfaction in the coming months. The report will reveal sales trends, best-selling products, and customer buying behaviors to guide effective planning for the year ahead.
